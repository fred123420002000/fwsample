# fwsimple

# 沒有任何實際功能的一些文件
各種框架的上課筆記，記錄一些前置宣告、連結、使用方式、設定細節

* hibernate
	* hibernate_cfg_xml 資料庫Driver位址名稱密碼以及.hbm.xml的匹配設定
	* hibernate_MODEL_java 無關連情況下的資料表Model
	* XXX_DAO_java 以框架進行的資料庫存取
	* XXX_hbm_xml 對Model與資料表的對應設定
* noframework
	* servlet 在servlet進行的編碼設定session處理、頁面轉換
	* sql 基本sql語法的add query delete update作法
* spring
	* factory_MODEL 工廠模式
	* spring_MODEL_java 一般模式
	* spring_xml 使用bean的id class建構式參數、初始值設定
	* use_spring_java 使用getBean的範例
* struts
	* action 對應的頁面訊息處裡
	* struts_xml 對應的頁面轉換
	* web_xml 需要的過濾器類別與頁面指定
